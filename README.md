Trabalho IV: Portal ongplus.com para Administração de ONGS em HTML5, CSS3 e JavaScript - Universidade Cruzeiro do Sul Desenvolvimento Front-End Para Web

O portal ONGPLUS é um portal de administração de ongs composto por três páginas Início, Projetos e Cadastro de Voluntário, com o intuito de exercitar o desenvolvimento front-end utilizando as tags semânticas do HTML5, como: header, footer, nav, main, article, video, address. E atributos em campos de formulário como required e placeholder, type email e tel que auxiliam na validação de formulários de modo nativo.

A formatação foi desenvolvida em CSS 3 em separado do HTML, seguindo as boas práticas de codificação. Foram usados recursos modernos como grid e flexbox para a construção dos layouts, responsivo e variáveis para configuração dinâmica.

Na página Cadastro de Volutário, o formulário recebeu o auxílio de código JavaScript para a validação e cadastro. Foi utilizado declaração de variáveis como const, let e var, funções, manipulação de objetos como getElementById(), dentre outros.

Projeto codificado para ter acessibilidade, com CSS e JavaScript minificados e histórico de commits.  

Tecnologias Utilizadas: HTML 5; css 3 e JavaScript.
